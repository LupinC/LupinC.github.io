{project_bookbot_temp}

The main purpose of this project is for the user to upload textbooks as pdfs and then ask related question for the bookbot to answer. If it were to be deployed locally for personal use, one should use the llama2 as the llm, otherwise, using the gpt 3.5 as llm is applicable as well. The frontend is built with javascript react; the backend is built with python and fast api framework. I was responsible for backend developemenet and project deployment.
